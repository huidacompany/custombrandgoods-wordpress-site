# Custom Brand Goods WordPress Site

This repository is for a custom WordPress B2B website project for Custom Brand Goods.

The project is intended for planning and development of a custom WordPress implementation, including a dedicated theme and site-specific plugin structure.

## Project Structure

- `wp-content/themes/custom-b2b-theme/` — custom B2B WordPress theme directory.
- `wp-content/plugins/site-core/` — custom site functionality plugin directory.
- `docs/` — planning and documentation files.

## Notes

- WooCommerce is not included.
- This repository is not connected to a production WordPress website.
- No deployment is configured.
