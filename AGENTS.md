# AGENTS.md

## Project Overview

This repository contains a custom WordPress B2B website project for Custom Brand Goods.

## Working Guidelines

- Do not connect this repository to a production WordPress website unless explicitly requested.
- Do not deploy the website unless explicitly requested.
- Do not add WooCommerce unless explicitly requested.
- Keep documentation in the `docs/` directory.
- Keep custom theme work under `wp-content/themes/custom-b2b-theme/`.
- Keep custom site functionality under `wp-content/plugins/site-core/`.

## Scope Notes

This project is currently in the setup and planning stage. Avoid building a full website until requirements are confirmed.
